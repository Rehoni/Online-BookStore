THE MAKER BUS

Prizes

1st 2nd 3rd

Der zweite Preis ist ein Arduino Roboter - der erste offizielle Arduino auf Rädern! Der Roboter hat zwei Prozessoren, je einer auf seinen beiden Platinen. Die Motor Platine steuert den Motor, und die Kontroll Platine empfängt die Signale von den Sensoren und entscheidet das Vorgehen. Jede der Platinen ist eine vollwertige Arduino Einheit, welche über die Arduino IDE programmiert wird.
